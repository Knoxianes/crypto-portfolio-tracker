## About
**Crypto portfolio tracker is an app for tracking your crypto coins,<br> buy and sell transactions and much more!**
### Currently supported coins:
- Bitcoin
- Ethereum
- USD Tether
- USD Coin
- BNB
- Solana
- Cardano
- Chainlink
- Polygon
- Polkadot
